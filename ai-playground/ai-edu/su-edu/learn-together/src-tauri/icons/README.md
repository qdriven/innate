# Tauri Icons

Place your application icons here with the following filenames:
- 32x32.png
- 128x128.png
- 128x128@2x.png
- icon.icns (macOS)
- icon.ico (Windows)

You can generate these icons using:
```bash
npm run tauri icon /path/to/your/source-image.png
```

The source image should be at least 1024x1024 pixels in PNG format.
