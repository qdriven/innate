# Susu 项目概述

## 项目名称
**Susu (苏苏)** - 互动学习平台

## 项目目标
创建一个让家长和孩子能够一起学习任何教育主题的互动学习平台。通过结合 3D 可视化、闪卡和测验功能，让学习变得更加有趣和有效。

## 核心理念
- **共同学习**: 让家长和孩子可以互相教学，共同探索知识
- **互动可视化**: 将抽象概念转化为直观的 3D/2D 可视化
- **游戏化学习**: 通过闪卡和测验增加学习的趣味性
- **离线优先**: 支持离线使用，数据存储在本地

## 目标用户
- 家长和孩子（主要用户）
- 教育工作者
- 自学者

## 平台支持
- **Web**: 通过浏览器访问
- **Desktop**: 可打包为桌面应用（未来支持 Tauri）
- **Mobile**: 响应式设计，支持移动端浏览器

## 功能模块

### 1. 可视化学习模块
- 3D 场景渲染（Three.js）
- 2D 图表可视化（SVG/D3.js）
- 交互控制面板
- 实时参数调整

### 2. 闪卡学习模块
- 翻卡交互
- 学习进度追踪
- 掌握度标记

### 3. 测验模块
- 多选题测验
- 即时反馈
- 成绩统计

### 4. 内容管理模块
- 主题浏览和搜索
- 新建/编辑主题
- 导入/导出数据

### 5. 更新模块
- 检查更新
- 下载新内容
- 版本管理

```
用户交互 → Zustand Store → 本地存储 (localStorage)
                    │
                    ▼
              API 服务 (可选)
                    │
                    ▼
              远程数据同步
```

## 项目来源

本项目结合了两个开源项目的特点：
1. **AetherViz Master** - 互动教育可视化工具，专注于 3D 交互式网页生成
2. **Gemini Flashcards Pro** - AI 驱动的闪卡学习工具
